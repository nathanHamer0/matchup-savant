Panda library:
    Data Frames
        df[df["col"] == "target"]
            - returns a refactored data frame consisting only of those rows for which under the specified column the scalar matches the target
            e.g., refactored_data_frame = df[df["pitch_type"] == "FF"]
    Series'
        df["col_header"]
            - returns a series of the scalars under the specified column
            e.g., scalar_series_under_col = df["pitch_type"]
        df.loc[df["col"] == "target", "col_header"]
            - same as:
                 refactored_data_frame = df[df["col"] == "target"] 
                 scalar_series_under_col_at_select_rows = refactored_data_frame["col_header"]
            - returns a series of scalars under the specified column for those which's rows pass the filter
            e.g., scalar_series_under_col_at_select_rows = df.loc[df["pitch_type"] == "FF", "run_value_per_100"]
    Scalars
        series.iloc[0]
            - returns the first scalar of the series
            e.g., scalar = df.loc[df["pitch_type"] == "FF", "run_value_per_100"].iloc[0]
    .loc
        - only necessary when a row is being selected
        - not necessary for column selection
    .iloc[i]
        - necessary for selecting from a series
        - unless series is already being traversed in a for-loop
